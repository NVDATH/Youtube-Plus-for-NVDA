# YoutubePlus для NVDA

> YoutubePlus — це додаток для користувачів NVDA, які люблять YouTube, але відчувають труднощі з доступом до багатьох функцій сайту — наприклад, читання коментарів, відстеження каналів або перегляд живого чату.
>
> Ми надаємо ці функції через інтерфейс NVDA у зручному для клавіатурної навігації вигляді з підтримкою гарячих клавіш і повним налаштуванням — **без потреби в API-ключах або підключенні облікового запису Google/YouTube**.
>
> Ви можете стежити за улюбленими каналами і бути впевнені, що побачите кожне відео з цих каналів — без фільтрації алгоритмом YouTube. Система «Вибране» дозволяє зберігати відео, канали, списки відтворення та список «Переглянути пізніше».
>
> Вбудований пошук відео відображає результати всередині інтерфейсу додатка — не просто відкриває YouTube у браузері. Функція завантаження відео/аудіо надається для зручності — якщо завантаження є вашою основною потребою, рекомендуємо спеціалізовані інструменти.
>
> Цей додаток **не** вбудовує відеопрогравач. Ми вважаємо, що вебпрогравач YouTube вже достатньо доступний. Якщо ви все ще відчуваєте труднощі, можна скористатися такими додатками, як [browserNav](https://addonstore.nvaccess.org/?channel=stable&language=en&apiVersion=2025.3.2&addonId=browsernav).

## Клавіатурні скорочення

Цей додаток використовує систему **шарових команд (Layer Commands)**, щоб уникнути конфліктів з іншими додатками або командами NVDA.

**Як використовувати:**
1. Натисніть **NVDA+Y**, щоб увійти в режим команд YoutubePlus (ви почуєте звуковий сигнал).
2. Натисніть відповідну літеру, щоб активувати потрібну функцію.

> **Примітка:** Якщо `NVDA+Y` конфліктує з іншим додатком, змінити прив'язку можна в `NVDA → Параметри → Жести вводу...` у категорії «YoutubePlus».

> **Визначення URL:** Для команд, яким потрібна URL-адреса відео, додаток спочатку перевіряє **поточне вікно браузера**. Якщо YouTube-посилання там не знайдено — автоматично перевіряє **буфер обміну**.

### Усі клавіші в режимі YoutubePlus

| Клавіша | Функція |
|---------|---------|
| **a** | Меню «Додати до...» (Вибране / Підписки) |
| **f** | Відкрити вибрані відео |
| **c** | Відкрити вибрані канали |
| **p** | Відкрити вибрані списки відтворення |
| **w** | Відкрити список «Переглянути пізніше» |
| **b** | Завантажити субтитри |
| **d** | Завантажити відео або аудіо |
| **e** | Пошук на YouTube |
| **i** | Показати інформацію про відео |
| **t** | Показати розділи / часові мітки |
| **m** | Керування підписками |
| **s** | Відкрити стрічку підписок |
| **u** | Менеджер профілів користувача |
| **l** | Показати коментарі або розпочати моніторинг живого чату |

| **y** | Відкрити налаштування YoutubePlus |
| **h** | Відкрити довідку |

## Детальний опис функцій

### a: Меню «Додати до...»

Відкриває підменю для вибору місця призначення:

- **Додати до вибраних відео** — зберігає поточне відео у Вибраному
- **Додати до вибраних каналів** — зберігає канал поточного відео у Вибраному
- **Додати до вибраних списків відтворення** — зберігає поточний список відтворення у Вибраному
- **Підписатися на канал** — стежить за каналом через систему підписок додатка
- **Додати до списку «Переглянути пізніше»** — зберігає поточне відео для перегляду пізніше

Більшість команд працюють з будь-яким форматом URL YouTube. Наприклад, якщо ви на сторінці відео і вибираєте «Додати до вибраних каналів», додаток автоматично витягує URL каналу.

**Виняток:** Для списків відтворення необхідно відкрити сторінку плейліста YouTube або мати коректну URL-адресу плейліста в буфері обміну.

---

### d: Завантаження відео/аудіо

Натисніть **NVDA+Y → D**, щоб відкрити діалог вибору формату:

- **Відеофайл (MP4)** — завантажує відео зі звуком
- **Аудіофайл (M4A)** — завантажує лише аудіо

Під час завантаження відображається діалог із відсотком прогресу. Кнопка **Скасувати** доступна в будь-який момент.

Папку призначення можна налаштувати в розділі [Налаштування](#налаштування).

> **Примітка:** Ця функція надається для зручності. Для масового завантаження рекомендуються спеціалізовані інструменти.

---

### b: Завантаження субтитрів

Натисніть **NVDA+Y → B**, щоб отримати список доступних мов субтитрів для поточного відео, після чого оберіть мову в діалозі. Субтитри поділяються на два типи:

- **(manual)** — субтитри, створені автором відео або спільнотою
- **(auto)** — субтитри, автоматично згенеровані YouTube

Підтримувані формати файлів: **SRT, VTT, TTML** і **TXT** (звичайний текст без часових міток). Налаштовується в розділі [Налаштування](#налаштування).

---

### e: Пошук на YouTube

Натисніть **NVDA+Y → E**, щоб відкрити вікно пошуку. Введіть запит і натисніть Enter для негайного пошуку. Натисніть Tab, щоб налаштувати кількість результатів (додаток запам'ятовує це значення для наступного разу).

Результати відображаються у форматі [Список відео](#список-відео), що використовується у всьому додатку — не у вигляді сторінки YouTube в браузері. Результати можуть включати відео, канали та списки відтворення.

---

### i: Інформація про відео

Натисніть **NVDA+Y → I**, щоб переглянути такі відомості про поточне відео:

- Назва
- Канал
- Тривалість
- Дата завантаження
- Кількість переглядів
- Кількість вподобань
- Кількість коментарів
- Опис

---

### t: Розділи / Часові мітки

Натисніть **NVDA+Y → T**, щоб переглянути список розділів або часових міток відео (якщо автор їх додав). Якщо додаток повідомляє «Розділів не знайдено», відео просто не містить цих даних.

Це вікно включає:

- **Поле пошуку** — фільтрує список розділів у реальному часі; Enter не потрібен
- **Список розділів** — показує назву розділу та час початку
- **Текстова область** — відображає назву вибраного розділу в зручному для читання вигляді
- **Кнопка «Відкрити розділ»** (або натисніть Space/Enter) — переходить до цього розділу у браузері
- **Кнопка «Копіювати назву»** — копіює назву розділу
- **Кнопка «Копіювати URL»** — копіює URL з часовою міткою цього розділу
- **Кнопка «Експортувати»** — зберігає всі розділи у текстовий файл

### l: Коментарі / Живий чат / Запис чату

YoutubePlus підтримує три типи вмісту через цю команду:

#### 1. Коментарі (для звичайних відео)

Натисніть **NVDA+Y → L** на сторінці відео. Додаток завантажить усі коментарі. Закріплені коментарі відображаються першими, потім усі інші в порядку, налаштованому в Параметрах.

**Вікно коментарів включає:**

- **Поле пошуку** — фільтрує коментарі в реальному часі
- **Поле фільтра** — попередньо налаштовані фільтри:
  - Без фільтра — показати всі
  - Фільтр за вибраним автором — показати лише коментарі від обраного користувача
  - Лише Super Chats
  - Лише Super Stickers
  - Лише Super Thanks
- **Список коментарів** — показує ім'я автора та текст; підтримується відображення відповідей у вигляді гілки
- **Текстова область (тільки читання)** — показує повний текст вибраного коментаря (зручно для довгих коментарів)
- **Кнопка «Копіювати»** (Alt+C або Ctrl+C) — копіює вибраний коментар
- **Кнопка «Експортувати»** (Alt+E) — зберігає всі коментарі у текстовий файл

#### 2. Живий чат (для активних трансляцій)

Для відео, що транслюється в прямому ефірі, натисніть L, щоб відкрити вікно отримання повідомлень живого чату. Відображаються лише повідомлення, отримані після активації команди — попередня історія не захоплюється.

- Закрийте і знову відкрийте вікно командою **V**, поки моніторинг не зупинено.
- Використовуйте команду **R** для перемикання автоматичного читання нових повідомлень — підходить для трансляцій з нечастими повідомленнями. Для трансляцій з великою кількістю повідомлень рекомендується вимкнути автоматичне читання і прокручувати вікно вручну.
- Натисніть **Shift+L**, щоб зупинити моніторинг. Після зупинки додаток запитає, чи бажаєте ви зберегти історію чату у файл.

**Пов'язані параметри:**
- **Автоматично зачитувати живий чат:** Читає нові повідомлення вголос одразу при надходженні (аналог команди R, але зберігається як стандартне налаштування).
- **Інтервал оновлення живого чату:** Як часто (у секундах) додаток перевіряє нові повідомлення (за замовчуванням: 5 секунд).
- **Ліміт історії повідомлень:** Максимальна кількість повідомлень, що зберігаються в пам'яті (за замовчуванням: 5 000). Жорстке обмеження становить 200 000 повідомлень для запобігання надмірного використання пам'яті.

#### 3. Запис живого чату (для архівних трансляцій)

Для відео, що раніше транслювалося в прямому ефірі, якщо канал не видалив чат, натискання L відкриє діалог із запитом: переглянути **Коментарі** чи **Запис живого чату**. Вікно запису чату має таку саму структуру, як і вікно коментарів, але доповнено:

- **Загальна сума донатів** — показує загальну суму пожертвувань (Super Chats / Super Stickers) під час трансляції

### Вибране (f, c, p, w)

Вікно «Вибране» поділено на 4 вкладки, доступні через окремі команди або всі разом в одному вікні.

| Клавіша | Вкладка |
|---------|---------|
| **F** | Збережені відео (Вибрані відео) |
| **C** | Збережені канали (Вибрані канали) |
| **P** | Збережені списки відтворення (Вибрані плейлісти) |
| **W** | Відео для перегляду пізніше (Список «Переглянути пізніше») |

#### Гарячі клавіші у вікні «Вибране»

- **Ctrl+1 до Ctrl+4** — перемикання між вкладками
- **Ctrl+Up/Down** — зміна порядку вкладок
- **Ctrl+C / Ctrl+X / Ctrl+V** — копіювати/вирізати/вставити для переміщення елементів
  _(Вибрані відео ↔ Список «Переглянути пізніше» можна переміщувати між собою; Канали та Плейлісти переміщуються лише всередині свого списку)_
- **F2** — перейменувати вибране відео/канал/плейліст вручну
- **Alt+R або Delete** — видалити вибраний елемент
- **Alt+N** — додати новий елемент з URL у буфері обміну
- **Alt+S** — перенести фокус до поля пошуку (результати оновлюються в реальному часі)
- **Кнопка «Сортування...» (Alt+O)** — сортування списку:
  - Сортувати за: Назвою, Каналом, Тривалістю, Датою додавання або Датою завантаження
  - Вибрати за зростанням або спаданням
  - Якщо встановлено «Застосувати постійно», порядок зберігається; інакше скидається під час пошуку або оновлення
  - Натисніть «Очистити сортування», щоб відновити початковий порядок

#### Вкладка «Канали» (Вибране)

Ця вкладка пропонує більше, ніж простий список — вона також включає:
- **Текстова область з описом каналу** — показує біографію/інформацію про канал
- **Кнопка відкриття каналу в браузері**
- **Кнопки перегляду Відео / Shorts / Live** цього каналу безпосередньо

#### Вкладка «Плейлісти» (Вибране)

- Натисніть **Space, Enter або Alt+V** — розгорнути всі відео в плейлісті
- **Кнопка «Відкрити в браузері» (Alt+W)** — відкриває плейліст у браузері

### s: Стрічка підписок

Вікно з відео від каналів, на які ви підписані через додаток. Це **не пов'язано** з вашим обліковим записом YouTube — вхід в систему не потрібен.

Стандартний вигляд має 4 вкладки за типом вмісту:

| Вкладка | Вміст |
|---------|-------|
| **Всі** | Усі типи вмісту разом |
| **Відео** | Лише звичайні відео |
| **Shorts** | Лише короткі відео |
| **Live** | Прямі трансляції та їх записи |

Ви також можете створювати **власні категорії** і налаштовувати, які канали відображаються в кожній.

#### Гарячі клавіші в стрічці підписок

- **Ctrl+1 до Ctrl+0** — перехід до вкладки категорії (до 10 вкладок)
- **Ctrl+Up/Down** — зміна порядку вкладок/категорій
- **F2** — перейменувати категорію (крім 4 стандартних вкладок)
- **Ctrl+= (Equals)** — додати нову категорію
- **Ctrl+- (Minus)** — видалити категорію (крім 4 стандартних вкладок)
- **Delete або Alt+S** — позначити відео як переглянуте; воно зникне зі списку
- **Ctrl+Delete** — позначити всі відео в поточній вкладці як переглянуті

#### Кнопки у вікні стрічки підписок

- **Позначити як переглянуте (Alt+S)** — позначає вибране відео як переглянуте
- **Додати нову підписку з буфера обміну (Alt+N)** — підписується на канал за URL у буфері обміну
- **Оновити стрічку (Alt+U)** — вручну запускає оновлення від усіх підписаних каналів (додаток також автоматично оновлюється при запуску NVDA)
- **Більше... (Alt+M)** — додаткові параметри:
  - Позначити всі в поточній вкладці як переглянуті (Ctrl+Delete)
  - Показати всі відео (включаючи переглянуті) — перемикає між «тільки непереглянуті» та «всі»; налаштування зберігається автоматично
  - Керування підписками...
  - Додати нову категорію... (Ctrl+=)
  - Перейменувати поточну категорію... (F2)
  - Видалити поточну категорію... (Ctrl+-)
  - **Очистити всі відео стрічки...** — видаляє всі відео з бази даних, не видаляючи підписані канали; корисно, якщо база даних занадто розрослася і впливає на продуктивність NVDA

### m: Керування підписками

Вікно з усіма каналами, на які ви підписані, та параметрами керування кожним:

- **Фільтр за категорією** — фільтрує список каналів за категорією (за замовчуванням: Всі)
- **Призначити категорії** — вибирає, у яких категоріях відображатиметься вміст цього каналу
- **Типи вмісту для завантаження** — вибирає, які типи вмісту оновлювати для цього каналу (Відео, Shorts, Live); корисно для каналів, що публікують лише певні типи
- **Переглянути вміст... (Alt+C)** — переглядає вміст каналу (аналогічно кнопці «Дія»)
- **Додати нову підписку з буфера обміну... (Alt+N)** — підписується на новий канал за URL у буфері обміну
- **Відписатися від каналу (Alt+U)** — видаляє канал з підписок
- **Зберегти зміни** — ⚠️ **Важливо:** необхідно натиснути перед закриттям вікна, інакше зміни не будуть збережені

### u: Менеджер профілів користувача

Додаток підтримує кілька **профілів користувача** на одному комп'ютері. Кожен профіль зберігає свої дані повністю окремо (Вибране, Підписки, Список «Переглянути пізніше»).

У цьому вікні:
- **F2** — перейменувати вибраний профіль
- **Delete** — видалити вибраний профіль ⚠️ Видалення є незворотним; усі дані в цьому профілі будуть втрачені

Щоб змінити профіль, перейдіть до [Налаштування](#налаштування) → Активний профіль, потім перезапустіть NVDA.

### Список відео

Список відео — стандартний інтерфейс, що використовується у всьому додатку: у результатах пошуку, Вибраному, стрічці підписок і при перегляді відео каналу.

- Натисніть **Enter**, щоб відкрити відео в браузері
- Натисніть **Space**, щоб виконати швидку дію (налаштовується в Параметрах)

Перейдіть до `NVDA → Параметри → Налаштування...` і виберіть категорію **«YoutubePlus»**.

| Параметр | Опис |
|----------|------|

| **Керування профілями** | Відкриває менеджер профілів користувача |

| **Режим сповіщень** | Як додаток сигналізує про фонову активність: **Сигнал** (короткі тони), **Звук** (аудіофайл), **Тихий** (без аудіо, але голосові повідомлення залишаються) |
| **Порядок сортування за замовчуванням** | Стандартний порядок відображення: **Спочатку нові** або **Спочатку старі** — застосовується до коментарів, чату та списків відео каналу |

| **Типи вмісту за замовчуванням** | Типи вмісту для завантаження при додаванні нових підписок: Відео, Shorts, Live |
| **Інтервал фонового оновлення** | Як часто додаток автоматично перевіряє новий вміст від підписаних каналів (вимкнено або від 15 хвилин до 24 годин) |
| **Автоматично зачитувати живий чат** | Читає нові повідомлення живого чату вголос при надходженні |
| **Інтервал оновлення живого чату** | Як часто (у секундах) перевіряються нові повідомлення (за замовчуванням: 5 секунд) |
| **Ліміт історії повідомлень** | Максимальна кількість повідомлень чату в пам'яті під час сесії (за замовчуванням: 5 000) |
| **Метод Cookie (Експериментально)** | Виберіть браузер, в якому ви авторизовані на YouTube. Додаток витягне Cookie з цього браузера для автентифікації запитів, що може допомогти вирішити помилку «Sign in to confirm you're not a bot». Зверніть увагу: ця функція є експериментальною, і результати залежать від браузера та конфігурації системи. |
| **Формат субтитрів за замовчуванням** | Формат файлу субтитрів: SRT, VTT, TTML або TXT (звичайний текст без часових міток) |
| **Папка завантажень та експорту** | Папка призначення для завантажених відео, аудіо та експортованих файлів |
| **Зробити резервну копію зараз** | Негайно створює резервну копію всіх даних активного профілю (додаток також виконує автоматичне щоденне резервне копіювання) |
| **Відновити дані з резервної копії** | Показує доступні резервні копії (до останніх 5 днів) для вибору та відновлення |

## Додаткова інформація

Цей додаток використовує дві основні бібліотеки: [pytchat](https://pypi.org/project/pytchat/) для моніторингу живого чату та [yt-dlp](https://pypi.org/project/yt-dlp/) для доступу до всіх інших даних YouTube. Висловлюємо щиру подяку розробникам обох бібліотек.

### Про yt-dlp

[yt-dlp](https://github.com/yt-dlp/yt-dlp) — один із найпотужніших інструментів з відкритим вихідним кодом для завантаження відео та аудіо з вебсайтів по всьому світу — підтримує понад 1 000 сайтів, а не лише YouTube. Він безкоштовний, з відкритим вихідним кодом, активно підтримується глобальною спільнотою і не містить реклами чи шкідливого програмного забезпечення.

**Рекомендації щодо використання:**

1. **Добросовісне використання:** Уникайте завантаження великих обсягів даних або частих запитів за короткий час. YouTube може виявити незвичайну активність і тимчасово обмежити доступ з вашої IP-адреси.
2. **Авторські права та конфіденційність:** Будь-які отримані дані або вміст слід використовувати лише для особистого перегляду або аналізу. Будь ласка, дотримуйтесь Умов використання кожної платформи і не використовуйте дані у спосіб, що порушує авторські права.
3. **Відповідальність:** Ви несете відповідальність за використання цього програмного забезпечення. Розробник додатка надає лише інтерфейс для доступу до даних YouTube через бібліотеку yt-dlp.

> **Порада:** Якщо вам потрібно обробити великі обсяги даних, розподіліть запити в часі для підтримки стабільності з'єднання та уникнення обмежень доступу.
