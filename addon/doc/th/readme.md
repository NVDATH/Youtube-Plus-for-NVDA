# YoutubePlus for NVDA

> YoutubePlus เป็นส่วนเสริมสำหรับผู้ใช้ NVDA ที่รักการใช้ YouTube แต่พบว่าฟีเจอร์หลายอย่างบนเว็บไซต์เข้าถึงได้ยาก เช่น การอ่านคอมเมนต์ การติดตามช่อง หรือการดูแชทสด
>
> เราเอาฟีเจอร์เหล่านี้มาแสดงผ่านส่วนติดต่อผู้ใช้ของ NVDA ในรูปแบบที่นำทางด้วยคีย์บอร์ดได้ง่าย รองรับทางลัด และปรับแต่งได้เต็มที่ — **โดยไม่ต้องใช้ API Key หรือเชื่อมต่อบัญชี Google/YouTube ใด ๆ ทั้งสิ้น**
>
> คุณสามารถติดตามช่องที่ชอบและมั่นใจได้ว่าจะเห็นทุกวิดีโอจากช่องเหล่านั้น โดยไม่ถูก Algorithm ของ YouTube กรองออก นอกจากนี้ยังมีระบบ Favorites สำหรับวิดีโอ ช่อง เพลย์ลิสต์ และ Watch List เพื่อบันทึกเนื้อหาที่สนใจแต่ยังไม่มีเวลาดู
>
> มีระบบค้นหาวิดีโอในตัว ซึ่งแสดงผลลัพธ์ภายใน UI ของส่วนเสริมเอง ไม่ใช่เพียงเปิดเบราว์เซอร์ไปยัง YouTube และมีฟีเจอร์ดาวน์โหลดวิดีโอ/เสียง ซึ่งให้มาเพื่อความสะดวก — หากต้องการดาวน์โหลดเป็นหลัก แนะนำให้ดูส่วนเสริมอื่นที่เน้นฟีเจอร์นี้โดยเฉพาะ
>
> สิ่งที่ส่วนเสริมนี้ **ไม่ทำ** คือการฝัง Video Player เพราะเราเชื่อว่าเว็บเพลเยอร์ของ YouTube นั้นเข้าถึงได้ดีพออยู่แล้ว หากยังพบปัญหา สามารถใช้ส่วนเสริมอื่นเช่น [browserNav](https://addonstore.nvaccess.org/?channel=stable&language=en&apiVersion=2025.3.2&addonId=browsernav) เพื่อปรับปรุงประสบการณ์การใช้งาน

## ระบบคีย์ลัด

ส่วนเสริมนี้ใช้ระบบ **Layer Command** เพื่อหลีกเลี่ยงการชนกับคีย์ลัดของส่วนเสริมหรือ NVDA อื่น ๆ

**วิธีใช้:**
1. กด **NVDA+Y** เพื่อเข้าสู่ YoutubePlus Command Mode (จะได้ยินเสียงแจ้งเตือน)
2. กดตัวอักษรที่ต้องการเพื่อเรียกฟีเจอร์นั้น ๆ

> **หมายเหตุ:** หาก `NVDA+Y` ชนกับส่วนเสริมอื่น สามารถเปลี่ยนได้ที่ `NVDA → Preferences → Input Gestures...` ในหมวด "YoutubePlus"

> **การค้นหา URL:** สำหรับคำสั่งที่ต้องใช้ URL วิดีโอ ส่วนเสริมจะตรวจสอบจาก **หน้าเบราว์เซอร์ที่เปิดอยู่** ก่อน หากไม่พบจึงตรวจสอบจาก **Clipboard** โดยอัตโนมัติ

## รายละเอียดฟีเจอร์ทั้งหมด

### a: เมนูเพิ่มข้อมูล (Add to...)

เปิดเมนูย่อยให้เลือกว่าจะเพิ่มวิดีโอหรือช่องไปยังที่ใด:

- **Add to Favorite Videos** — บันทึกวิดีโอปัจจุบันลงใน Favorites
- **Add to Favorite Channels** — บันทึกช่องของวิดีโอปัจจุบันลงใน Favorites
- **Add to Favorite Playlists** — บันทึกเพลย์ลิสต์ปัจจุบันลงใน Favorites
- **Subscribe to Channel** — ติดตามช่องผ่านระบบ Subscription ของส่วนเสริม
- **Add to Watch List** — บันทึกวิดีโอปัจจุบันไว้ใน Watch List เพื่อดูทีหลัง

คำสั่งส่วนใหญ่ทำงานได้กับ URL ทุกรูปแบบของ YouTube — ตัวอย่างเช่น ถ้าคุณอยู่ที่หน้าวิดีโอและเลือก "Add to Favorite Channels" ส่วนเสริมจะดึง URL ของช่องมาให้โดยอัตโนมัติ

**ข้อยกเว้น:** สำหรับ Playlist ต้องเปิดหน้า YouTube Playlist หรือมี URL ของ Playlist อยู่ใน Clipboard

---

### d: ดาวน์โหลดวิดีโอ/เสียง

กด **NVDA+Y → D** จะเปิดกล่องโต้ตอบถามว่าต้องการดาวน์โหลดเป็น:

- **ไฟล์วิดีโอ (MP4)** — ดาวน์โหลดวิดีโอพร้อมเสียง
- **ไฟล์เสียง (M4A)** — ดาวน์โหลดเฉพาะเสียง

ระหว่างดาวน์โหลดจะมีแถบแสดงความคืบหน้าพร้อมตัวเลขเปอร์เซ็นต์ มีปุ่ม **Cancel** เพื่อยกเลิกได้ทุกเมื่อ

ตั้งค่าโฟลเดอร์ปลายทางได้ใน [Settings](#การตั้งค่า-settings)

> **หมายเหตุ:** ฟีเจอร์นี้ให้มาเพื่อความสะดวก หากต้องการดาวน์โหลดจำนวนมาก แนะนำให้ใช้เครื่องมือที่ออกแบบมาสำหรับงานนี้โดยเฉพาะ

---

### b: ดาวน์โหลดคำบรรยาย (Subtitle)

กด **NVDA+Y → B** จะดึงรายการภาษาคำบรรยายที่มีในวิดีโอนั้น แล้วแสดงกล่องโต้ตอบให้เลือกภาษา รายการจะแบ่งเป็นสองประเภท:

- **(manual)** — คำบรรยายที่ผู้สร้างเพิ่มเองหรือชุมชนเพิ่มให้
- **(auto)** — คำบรรยายที่ YouTube สร้างอัตโนมัติ

รูปแบบไฟล์ที่รองรับ: **SRT, VTT, TTML** และ **TXT** (ข้อความล้วน ไม่มี Timecode) ตั้งค่าได้ใน [Settings](#การตั้งค่า-settings)

---

### e: ค้นหาวิดีโอ

กด **NVDA+Y → E** เพื่อเปิดหน้าต่างค้นหา พิมพ์คำค้นหาแล้วกด Enter เพื่อค้นทันที กด Tab เพื่อปรับจำนวนผลลัพธ์ที่ต้องการ (ส่วนเสริมจดจำค่านี้ไว้สำหรับครั้งถัดไป)

ผลลัพธ์จะแสดงในรูปแบบ [รายการวิดีโอ](#รูปแบบรายการวิดีโอ-video-list) เดียวกับที่ใช้ในส่วนอื่น ๆ ของส่วนเสริม — ไม่ใช่การเปิด YouTube ในเบราว์เซอร์ ผลการค้นหาอาจรวมทั้งวิดีโอ ช่อง และเพลย์ลิสต์

---

### i: ข้อมูลวิดีโอ

กด **NVDA+Y → I** เพื่อดูข้อมูลต่อไปนี้ของวิดีโอปัจจุบัน:

- ชื่อวิดีโอ
- ช่อง
- ความยาว
- วันที่อัปโหลด
- จำนวนวิว
- จำนวน Like
- จำนวนคอมเมนต์
- คำอธิบายวิดีโอ (Description)

---

### t: Chapters / Timestamps

กด **NVDA+Y → T** เพื่อดูรายการ Chapter หรือ Timestamp ของวิดีโอ (หากผู้สร้างวิดีโอได้ใส่ข้อมูลนี้ไว้) ถ้าส่วนเสริมแจ้งว่า "No chapters found" หมายความว่าวิดีโอนั้นไม่มีข้อมูล Chapter

หน้าต่างนี้ประกอบด้วย:

- **ช่องค้นหา** — กรองรายการ Chapter แบบ Real-time ไม่ต้องกด Enter
- **รายการ Chapter** — แสดงชื่อ Chapter และเวลาเริ่มต้น
- **กล่องข้อความ** — แสดงชื่อ Chapter ที่เลือกในรูปแบบอ่านง่าย
- **ปุ่ม Open Chapter** (หรือกด Space/Enter) — ข้ามไปยัง Chapter นั้นในเบราว์เซอร์โดยตรง
- **ปุ่ม Copy Title** — คัดลอกชื่อ Chapter
- **ปุ่ม Copy URL** — คัดลอก URL พร้อม Timestamp ของ Chapter นั้น
- **ปุ่ม Export** — บันทึก Chapters ทั้งหมดเป็นไฟล์ข้อความ

### l: แสดงคอมเมนต์ / แชทสด / แชทย้อนหลัง

YoutubePlus รองรับเนื้อหา 3 ประเภทผ่านคำสั่งนี้:

#### 1. คอมเมนต์ (สำหรับวิดีโอปกติ)

กด **NVDA+Y → L** ขณะอยู่ที่หน้าวิดีโอ ส่วนเสริมจะดึงคอมเมนต์ทั้งหมดมาแสดง คอมเมนต์ที่ปักหมุด (Pinned) จะแสดงก่อน ตามด้วยคอมเมนต์อื่น ๆ ตามลำดับที่ตั้งค่าไว้ใน Settings

**หน้าต่างคอมเมนต์ประกอบด้วย:**

- **ช่องค้นหา** — กรองคอมเมนต์แบบ Real-time
- **Filter combo box** — ตัวกรองสำเร็จรูป:
  - No Filter — แสดงทั้งหมด
  - Filter by Selected Author — แสดงเฉพาะคอมเมนต์จากผู้เขียนที่เลือก
  - Show Super Chats Only
  - Show Super Stickers Only
  - Show Super Thanks Only
- **รายการคอมเมนต์** — แสดงชื่อผู้เขียนและข้อความ รองรับการแสดงผล Reply แบบ Thread
- **กล่องข้อความ (Read-only)** — อ่านข้อความคอมเมนต์ฉบับเต็มสำหรับความคิดเห็นที่ยาวมาก
- **ปุ่ม Copy** (Alt+C หรือ Ctrl+C) — คัดลอกคอมเมนต์ที่เลือก
- **ปุ่ม Export** (Alt+E) — บันทึกคอมเมนต์ทั้งหมดเป็นไฟล์ข้อความ

#### 2. แชทสด (Live Chat)

สำหรับวิดีโอที่กำลัง Live อยู่ กด L เพื่อเปิดหน้าต่างรับข้อความแชทสด ข้อความที่แสดงจะเป็นเฉพาะที่รับมาหลังจากเปิดคำสั่งนี้เท่านั้น (ไม่มีประวัติก่อนหน้า)

- ปิดหน้าต่างแล้วเปิดใหม่ได้ด้วยคำสั่ง **V** ตราบที่ยังไม่ได้หยุดมอนิเตอร์
- ใช้คำสั่ง **R** เพื่อสลับการอ่านข้อความใหม่อัตโนมัติ — เหมาะสำหรับ Stream ที่มีข้อความไม่บ่อยนัก สำหรับ Stream ที่มีข้อความมาก แนะนำให้ปิด Auto-read แล้วเลื่อนดูเองในหน้าต่าง
- ใช้คำสั่ง **Shift+L** เพื่อหยุดมอนิเตอร์ เมื่อหยุดแล้ว ส่วนเสริมจะถามว่าต้องการบันทึกประวัติแชทเป็นไฟล์หรือไม่

**การตั้งค่าที่เกี่ยวข้อง:**
- **Automatically speak incoming live chat:** อ่านข้อความใหม่อัตโนมัติ (เทียบเท่าคำสั่ง R แต่บันทึกเป็นค่า Default)
- **Live chat refresh interval:** ระยะเวลา (วินาที) ที่ส่วนเสริมตรวจข้อความใหม่ (Default: 5 วินาที)
- **Message history limit:** จำนวนข้อความสูงสุดที่เก็บไว้ในหน่วยความจำ (Default: 5,000 ข้อความ) ส่วนเสริมมีฝาครอบสูงสุดที่ 200,000 ข้อความเพื่อป้องกันการใช้หน่วยความจำเกิน

#### 3. แชทย้อนหลัง (Live Chat Replay)

สำหรับวิดีโอที่เคย Live มาแล้ว หากช่องไม่ได้ลบแชทออก กด L จะมีกล่องโต้ตอบถามว่าต้องการดู **คอมเมนต์** หรือ **แชทย้อนหลัง** หน้าต่างแชทย้อนหลังมีโครงสร้างเดียวกับหน้าต่างคอมเมนต์ แต่เพิ่มมาด้วย:

- **Total Paid Amount** — ยอดรวมการบริจาคทั้งหมด (Super Chat / Super Sticker) ในระหว่าง Stream

### Favorites (f, c, p, w)

หน้าต่าง Favorites แบ่งออกเป็น 4 แท็บ เข้าถึงได้ด้วยคำสั่งแยกกัน หรือเปิดรวมในหน้าต่างเดียวกัน

| คีย์ | แท็บ |
|------|------|
| **F** | วิดีโอที่บันทึกไว้ (Favorite Videos) |
| **C** | ช่องที่บันทึกไว้ (Favorite Channels) |
| **P** | เพลย์ลิสต์ที่บันทึกไว้ (Favorite Playlists) |
| **W** | รายการที่จะดูทีหลัง (Watch List) |

#### คีย์ลัดในหน้าต่าง Favorites

- **Ctrl+1 ถึง Ctrl+4** — สลับระหว่างแท็บ
- **Ctrl+Up/Down** — เรียงลำดับแท็บ
- **Ctrl+C / Ctrl+X / Ctrl+V** — คัดลอก/ตัด/วาง เพื่อย้ายรายการ  
  _(Favorite Videos ↔ Watch List สามารถย้ายข้ามกันได้ ส่วน Channels และ Playlists ย้ายได้เฉพาะภายในรายการตัวเอง)_
- **F2** — เปลี่ยนชื่อวิดีโอ/ช่อง/เพลย์ลิสต์ที่เลือกด้วยตนเอง
- **Alt+R หรือ Delete** — ลบรายการที่เลือก
- **Alt+N** — เพิ่มรายการใหม่จาก URL ใน Clipboard
- **Alt+S** — เลื่อนโฟกัสไปยังช่องค้นหา (ผลลัพธ์อัปเดตแบบ Real-time)
- **ปุ่ม Sort... (Alt+O)** — จัดเรียงรายการ:
  - เรียงตาม: ชื่อ, ช่อง, ความยาว, วันที่เพิ่ม, วันที่อัปโหลด
  - เลือก Ascending หรือ Descending
  - หากเลือก "Apply permanently" การเรียงจะถูกบันทึกถาวร มิฉะนั้นจะรีเซ็ตเมื่อค้นหาหรือรีเฟรช
  - กด "Clear Sort" เพื่อกลับสู่ลำดับเดิม

#### แท็บ Channel (Favorites)

แท็บนี้พิเศษกว่าแท็บอื่น — นอกจากรายการช่องแล้ว ยังมี:
- **กล่องข้อความคำอธิบายช่อง** — แสดงข้อมูล Bio ของช่อง
- **ปุ่มเปิดช่องในเบราว์เซอร์**
- **ปุ่มดูวิดีโอ/Shorts/Live** ของช่องนั้นโดยตรง

#### แท็บ Playlist (Favorites)

- กด **Space, Enter หรือ Alt+V** — ขยายรายการวิดีโอทั้งหมดในเพลย์ลิสต์
- **ปุ่ม Open on Web (Alt+W)** — เปิดเพลย์ลิสต์ในเบราว์เซอร์

### s: Subscription Feed

หน้าต่างแสดงวิดีโอจากช่องที่คุณติดตามผ่านส่วนเสริม (**ไม่ใช่** บัญชี YouTube ของคุณ ไม่ต้องล็อกอิน)

แท็บเริ่มต้นมี 4 แท็บตามประเภทเนื้อหา:

| แท็บ | เนื้อหา |
|------|---------|
| **All** | เนื้อหาทุกประเภทรวมกัน |
| **Video** | วิดีโอปกติ |
| **Shorts** | วิดีโอสั้น |
| **Live** | Live stream และ Live replay |

นอกจากนี้ยังสามารถสร้าง **หมวดหมู่แบบกำหนดเอง** และกำหนดว่าช่องใดจะแสดงในหมวดหมู่ใด

#### คีย์ลัดใน Subscription Feed

- **Ctrl+1 ถึง Ctrl+0** — ข้ามไปยังแท็บที่ต้องการ (รองรับสูงสุด 10 แท็บ)
- **Ctrl+Up/Down** — เรียงลำดับแท็บ/หมวดหมู่
- **F2** — เปลี่ยนชื่อหมวดหมู่ (ยกเว้น 4 แท็บเริ่มต้น)
- **Ctrl+= (Equals)** — เพิ่มหมวดหมู่ใหม่
- **Ctrl+- (Minus)** — ลบหมวดหมู่ (ยกเว้น 4 แท็บเริ่มต้น)
- **Delete หรือ Alt+S** — ทำเครื่องหมายว่าดูแล้ว (Seen) วิดีโอนั้นจะหายไปจากรายการ
- **Ctrl+Delete** — ทำเครื่องหมายว่าดูแล้วทั้งหมดในแท็บปัจจุบัน

#### ปุ่มในหน้าต่าง Subscription Feed

- **Mark as seen (Alt+S)** — ทำเครื่องหมายว่าดูแล้ว
- **Add new Subscription from clipboard URL (Alt+N)** — เพิ่มช่องใหม่จาก URL ใน Clipboard
- **Update Feed (Alt+U)** — รีเฟรชข้อมูลจากทุกช่องที่ติดตาม (ส่วนเสริมอัปเดตอัตโนมัติเมื่อ NVDA เริ่มทำงาน)
- **More... (Alt+M)** — เมนูเพิ่มเติม:
  - Mark all in current tab as seen (Ctrl+Delete)
  - Show all videos (รวมที่ดูแล้ว) — สลับระหว่างดูเฉพาะที่ยังไม่ดู กับ ดูทั้งหมด
  - Manage subscriptions...
  - Add New Category... (Ctrl+=)
  - Rename Current Category... (F2)
  - Remove Current Category... (Ctrl+-)
  - **Clear All Feed Videos...** — ลบวิดีโอทั้งหมดออกจากฐานข้อมูล โดยไม่ลบรายชื่อช่องที่ติดตาม เหมาะสำหรับเมื่อฐานข้อมูลโตมากจนกระทบประสิทธิภาพ NVDA

### m: จัดการ Subscription

หน้าต่างแสดงช่องทั้งหมดที่คุณติดตาม พร้อมตัวเลือกจัดการแต่ละช่อง:

- **Filter by Category** — กรองรายการช่องตามหมวดหมู่ (Default: All)
- **Assign to Categories** — กำหนดว่าเนื้อหาของช่องนี้จะแสดงในหมวดหมู่ใด
- **Content Types to Fetch** — เลือกประเภทเนื้อหาที่จะอัปเดตสำหรับช่องนี้ (Videos, Shorts, Live) มีประโยชน์สำหรับช่องที่เผยแพร่เฉพาะบางประเภท
- **View Content... (Alt+C)** — ดูเนื้อหาของช่อง (เช่นเดียวกับปุ่ม Action)
- **Add new subscribe channel from Clipboard... (Alt+N)** — เพิ่มช่องใหม่จาก URL ใน Clipboard
- **Unsubscribe from this Channel (Alt+U)** — ยกเลิกการติดตามช่อง
- **Save Changes** — ⚠️ **สำคัญ:** ต้องกดปุ่มนี้ก่อนปิดหน้าต่าง มิฉะนั้นการเปลี่ยนแปลงจะไม่ถูกบันทึก

## การตั้งค่า (Settings)

เข้าถึงได้ที่ `NVDA → Preferences → Settings...` แล้วเลือกหมวด **"YoutubePlus"**

| การตั้งค่า | คำอธิบาย |
|-----------|---------|
| **Active Profile** | เลือกโปรไฟล์ที่ใช้งาน (ต้อง Restart NVDA หลังเปลี่ยน) |
| **Manage Profiles** | เปิด User Profile Manager |
| **Quick Action (Space bar)** | กำหนดว่า Space จะทำอะไรในหน้าต่างรายการวิดีโอ เลือกได้จากทุก Action ใน Action menu |
| **Notification mode** | รูปแบบแจ้งเตือนขณะทำงานในพื้นหลัง: **Beep** (เสียง Beep), **Sound** (ไฟล์เสียง), **Silent** (ไม่มีเสียง แต่ยังมีการอ่านข้อความ) |
| **Default sort order** | ลำดับการแสดงผลเริ่มต้น: **Newest First** หรือ **Oldest First** ใช้กับรายการคอมเมนต์, แชท และวิดีโอของช่อง |
| **Items to fetch** | จำนวนรายการที่ดึงต่อประเภทเนื้อหาเมื่อเปิดช่องหรืออัปเดต Feed (Default: 20, ช่วง 5–100) |
| **Default content types** | ประเภทเนื้อหาที่ดึงสำหรับช่องที่ติดตามใหม่: Videos, Shorts, Live |
| **Background update interval** | ความถี่ที่ส่วนเสริมตรวจหาวิดีโอใหม่โดยอัตโนมัติ (ปิด หรือ 15 นาที ถึง 24 ชั่วโมง) |
| **Automatically speak incoming live chat** | อ่านข้อความแชทสดใหม่ให้ฟังอัตโนมัติเมื่อมาถึง |
| **Live chat refresh interval** | ระยะเวลาตรวจข้อความใหม่ (Default: 5 วินาที) |
| **Message history limit** | จำนวนข้อความแชทสูงสุดที่เก็บไว้ในหน่วยความจำ (Default: 5,000) |
| **Cookie method (Experimental)** | เลือกเบราว์เซอร์ที่คุณล็อกอิน YouTube ไว้ ส่วนเสริมจะดึง Cookie จากเบราว์เซอร์นั้นมาใช้เพื่อพิสูจน์ตัวตน ซึ่งอาจช่วยแก้ปัญหา "Sign in to confirm you're not a bot" ได้ อย่างไรก็ตาม ฟีเจอร์นี้ยังอยู่ในขั้นทดลอง และผลลัพธ์ขึ้นอยู่กับเบราว์เซอร์และการตั้งค่าของแต่ละเครื่อง |
| **Default subtitle format** | รูปแบบไฟล์คำบรรยาย: SRT, VTT, TTML หรือ TXT (ข้อความล้วน ไม่มี Timecode) |
| **Default download and export folder path** | โฟลเดอร์ปลายทางสำหรับไฟล์ดาวน์โหลดและส่งออก |
| **Backup data now** | สำรองข้อมูลโปรไฟล์ปัจจุบันทันที (ส่วนเสริมสำรองให้อัตโนมัติทุกวัน) |
| **Restore data from backup** | เลือกคืนค่าข้อมูลจากไฟล์สำรองย้อนหลัง (เก็บไว้ 5 วันล่าสุด) |

## ข้อมูลเพิ่มเติม

ส่วนเสริมนี้ใช้ไลบรารีหลัก 2 ตัว ได้แก่ [pytchat](https://pypi.org/project/pytchat/) สำหรับมอนิเตอร์แชทสด และ [yt-dlp](https://pypi.org/project/yt-dlp/) สำหรับการเข้าถึงข้อมูล YouTube ทุกรูปแบบ ขอขอบคุณนักพัฒนาของทั้งสองไลบรารีอย่างสุดซึ้ง

### เกี่ยวกับ yt-dlp

[yt-dlp](https://github.com/yt-dlp/yt-dlp) เป็นเครื่องมือโอเพนซอร์สที่ทรงพลังสำหรับดาวน์โหลดวิดีโอและเสียงจากเว็บไซต์ทั่วโลก รองรับมากกว่า 1,000 เว็บไซต์ ฟรี ไม่มีโฆษณา และได้รับการดูแลอย่างต่อเนื่องโดยชุมชนนักพัฒนา

**แนวทางการใช้งานที่ควรทราบ:**

1. **ใช้งานอย่างเหมาะสม (Fair Use):** หลีกเลี่ยงการดึงข้อมูลจำนวนมากหรือส่งคำขอซ้ำ ๆ ในเวลาสั้น YouTube อาจตรวจพบและจำกัดการเข้าถึงจาก IP ของคุณชั่วคราว
2. **ลิขสิทธิ์และความเป็นส่วนตัว:** เนื้อหาที่ดึงมาควรใช้เพื่อชมส่วนตัวหรือวิเคราะห์เท่านั้น กรุณาเคารพข้อกำหนดการใช้งานของแต่ละแพลตฟอร์มและไม่นำไปใช้ในลักษณะที่ละเมิดลิขสิทธิ์
3. **ความรับผิดชอบ:** คุณเป็นผู้รับผิดชอบต่อการใช้งานซอฟต์แวร์นี้ นักพัฒนาส่วนเสริมเป็นผู้จัดเตรียมเพียงส่วนติดต่อสำหรับเข้าถึงข้อมูล YouTube ผ่านไลบรารี yt-dlp เท่านั้น

> **เคล็ดลับ:** หากต้องการประมวลผลข้อมูลจำนวนมาก ให้เว้นระยะห่างระหว่างคำขอแต่ละครั้ง เพื่อรักษาความเสถียรของการเชื่อมต่อและหลีกเลี่ยงการถูกจำกัดการเข้าถึง
